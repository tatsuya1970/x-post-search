## 実施手順

1. テスト担当者を1名以上確保
2. 上記チェックリストを印刷またはスプレッドシートで準備
3. 実際のアプリで各項目を確認
4. 結果列に ○ / △ / × を記入
5. 改善点があればコメント欄に記載

## 評価基準

- **○**: 合格基準を満たしている
- **△**: 一部満たしているが改善の余地あり
- **×**: 合格基準を満たしていない
